# SmartChannelX — بوت تيليجرام لإدارة القنوات

بوت تيليجرام احترافي مكتوب بلغة Python باستخدام `python-telegram-bot v21+` لإدارة قنوات تيليجرام بالكامل.

## ✨ الميزات

- 📝 **إنشاء منشورات** بأنواع مختلفة (نص، صورة، فيديو، ملف، صوت، وغيره) مع أزرار Inline.
- 📅 **جدولة منشورات** لمرة واحدة (تاريخ + وقت).
- 🔁 **منشورات متكررة** يومية / أسبوعية / شهرية مع مؤقت حذف اختياري.
- 🛡️ **حماية القناة**: كابتشا، فلاتر انضمام (حظر، أسماء، أبجدية)، فلتر كلمات، حظر البوتات تلقائيًا.
- 📊 **إحصائيات**: نمو المشتركين (مع رسم بياني)، تفاعل المنشورات، مشاهدات، أفضل المنشورات.
- ⚙️ **إعدادات القناة**: لغة، منطقة زمنية، توقيع تلقائي، إشعارات.
- 👤 **إدارة المشرفين** على مستوى البوت مع تحديد صلاحيات.
- 🌐 **تعدد اللغات** (عربي / إنجليزي) قابل للتغيير لكل قناة وللبوت ككل.
- 🛠️ **قائمة المطور** (`/dev_settings`) لإدارة مفاتيح API، التكاملات، السجلات، وإعادة التشغيل.
- 💾 **استعادة الجدولة بعد إعادة التشغيل** — كل المهام تُحفظ في قاعدة بيانات SQLite.

## 🚀 التشغيل

### المتطلبات

- Python 3.12+
- متغير بيئة: `TELEGRAM_BOT_TOKEN` (من @BotFather)
- متغيرات بيئة اختيارية:
  - `DEVELOPER_IDS` — معرفات المطورين (Telegram user IDs) مفصولة بفاصلة، للسماح بـ `/dev_settings`.
  - `SMARTCHANNELX_DB` — مسار قاعدة البيانات (افتراضي `./smartchannelx.db`).
  - `DEFAULT_LANGUAGE` — `ar` (افتراضي) أو `en`.
  - `DEFAULT_TIMEZONE` — مثل `Asia/Riyadh` (افتراضي).

### التشغيل المحلي

```bash
cd smartchannelx
python main.py
```

### إعداد البوت في تيليجرام

1. أنشئ بوتًا جديدًا عبر [@BotFather](https://t.me/BotFather) واحصل على التوكن.
2. أضف البوت كمشرف في قناتك مع صلاحيات النشر/التعديل/الحذف على الأقل.
3. ابدأ محادثة مع البوت بالضغط على `/start`.
4. اضغط ➕ إضافة قناة جديدة، ثم أرسل `@username` القناة أو وجّه أي رسالة منها.

## 🧱 البنية

```
smartchannelx/
├── main.py                 # نقطة الدخول
├── smartchannelx.db        # قاعدة بيانات SQLite (تُنشأ تلقائيًا)
└── bot/
    ├── config.py           # إعدادات + متغيرات بيئة
    ├── database.py         # طبقة SQLite غير متزامنة (aiosqlite)
    ├── i18n.py             # ترجمات (عربي/إنجليزي)
    ├── keyboards.py        # كل لوحات المفاتيح Inline
    ├── utils.py            # دوال مساعدة (تواريخ، أزرار، مناطق زمنية)
    ├── sender.py           # نشر المنشورات + مهام JobQueue
    ├── scheduler.py        # تسجيل/استعادة الجدولة
    └── handlers/
        ├── __init__.py     # تسجيل كل الـ handlers + موجه الرسائل
        ├── start.py        # ترحيب + مساعدة + تنقل عام
        ├── channels.py     # إضافة/قائمة/فتح القنوات
        ├── posts.py        # إنشاء منشور + أزرار + إعدادات + معاينة + نشر
        ├── schedule.py     # جدولة منشورات لمرة واحدة
        ├── recurring.py    # منشورات متكررة
        ├── protection.py   # حماية القناة + فلاتر
        ├── statistics.py   # إحصائيات + رسوم بيانية
        ├── settings.py     # إعدادات القناة
        ├── admins.py       # إدارة المشرفين
        ├── language.py     # تغيير القناة/اللغة
        └── developer.py    # قائمة المطور
```

## 🌍 النشر للإنتاج

البوت يستخدم **Long Polling** افتراضيًا (لا يحتاج Webhook ولا منفذ HTTP).
ببساطة اجعل `python main.py` يعمل بشكل دائم (workflow / systemd / pm2 / docker).

كل المنشورات المجدولة والمتكررة محفوظة في قاعدة البيانات وتُعاد جدولتها تلقائيًا عند التشغيل.
